# Aerodynamics Calculator Suite

A comprehensive aerodynamics calculator suite for aerospace engineering students and enthusiasts.

## Features

- **Lift Force Calculator** - Calculate lift generated by an airfoil
- **Drag Force Calculator** - Calculate aerodynamic drag
- **Lift-to-Drag Ratio** - Aerodynamic efficiency metric
- **Reynolds Number** - Flow regime classification
- **Mach Number** - Compressibility effects indicator
- **Wing Loading** - Aircraft performance metric
- **Stall Speed** - Minimum flight speed

## Installation

### Prerequisites

Formulas Used
Calculator	   Formula	Description
Lift	         L = ½ × ρ × V² × S × C_L	     Lift force in Newtons
Drag	         D = ½ × ρ × V² × S × C_D      Drag force in Newtons
L/D Ratio	     L/D = Lift / Drag	           Aerodynamic efficiency
Reynolds	     Re = ρ × V × L / μ	           Flow regime indicator
Mach	         M = V / a	                   Speed relative to sound
Wing Loading	 W/S = Weight / Area	         Load per wing area
Stall Speed	   V_s = √(2(W/S)/(ρ×C_Lmax))	   Minimum flight speed

Constants Used
Sea Level Air Density (ρ): 1.225 kg/m³
Air Viscosity (μ): 1.81 × 10⁻⁵ kg/(m·s)
Speed of Sound (a): 340.3 m/s
Gravitational Acceleration (g): 9.80665 m/s²
